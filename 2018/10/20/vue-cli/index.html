vue 脚手架之 vue-cli 的使用

2018-10-20
JavaScript, 脚手架

  vue 的发展是很快的，围绕 vue 已经有了一个完整的应用生态，比如 vue + vuex + vue-router + axios + webpack 就是一个完整的应用系统了。那么，每次我们要新建一个运用了此应用系统的项目的时候，都要重新搭建项目吗？回答是肯定的，也确实很麻烦，但是 vue-cli 作为一个 vue 的专门脚手架，可以轻松帮我们解决这样的问题。

  vue-cli 目前版本已经到 3.0 了。3.0 版本跟之前的 2.0 、1.0 这两个版本的使用方式已经不太一样了。为了跟上脚手架的发展，本篇文章着重介绍 3.0 版本的使用。
